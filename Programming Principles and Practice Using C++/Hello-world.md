# 🧠 The Classic First Program (Book-style explanation)

---

## 🧩 Program = Instructions (Recipe concept)

👉 ஒரு programன்னா:

**Computerக்கு கொடுக்குற exact instructions**

👉 Author comparison:

* 🍳 Cooking recipe
* 🧸 Toy assemble instructions

👉 அதே மாதிரி:
👉 **Computerக்கு step-by-step சொல்லுறது = program**

---

## 💻 Full Program

```cpp
// This program outputs the message "Hello, World!" to the monitor
import std;

int main()
{
    std::cout << "Hello, World!\n";
    return 0;
}
```

---

# 🔍 Line-by-line deep explanation

---

## 💬 1. Comment

```cpp
// This program outputs the message "Hello, World!"
```

👉 `//` → comment

👉 இது:

* compiler ignore பண்ணும்
* humanக்கு explain பண்ணும்

---

### 🎯 முக்கிய idea:

👉 Code இரண்டு audienceக்கு:

1. 🤖 Computer
2. 👨‍💻 Programmer

👉 அதனால்:
**Code readable ஆக இருக்கணும் 🔥**

---

## 📦 2. import std;

```cpp
import std;
```

👉 இது:

* standard library import பண்ணும்

---

### 🎯 Meaning:

👉 C++ already provide பண்ணும் tools:

* cout
* input/output
* many utilities

👉 அவை use பண்ண இதை import பண்ணணும்

---

## 🚀 3. main() function

```cpp
int main()
```

👉 இது மிகவும் முக்கியம் ⚠️

---

### 🎯 Rule:

👉 Every C++ program:

**main()ல இருந்து தான் start ஆகும்**

---

### 🧠 Function என்ன?

👉 Function =

**named instructions set**

---

### 🧩 Function parts:

1. `int` → return type
2. `main` → name
3. `()` → parameters
4. `{}` → body

---

## 🔥 Minimal program

```cpp
int main() { }
```

👉 இது valid
👉 ஆனால் useful இல்லை 😄

---

## 🖨️ 4. Output line (Most important)

```cpp
std::cout << "Hello, World!\n";
```

---

### 🔍 Breakdown:

---

### 🧵 "Hello, World!\n"

👉 இது:

* string literal
* double quotes `" "`

👉 `\n`:

👉 newline (next line move)

---

### 📺 cout

👉 `cout` = character output stream

👉 pronounce:
👉 “see-out”

👉 Meaning:
👉 screenக்கு output

---

### 🔗 << operator

👉 `<<` = output operator

👉 meaning:

👉 “data → coutக்கு அனுப்பு”

---

### 🧠 Full meaning:

👉
**“Hello, World!” print பண்ணு + next line போ”**

---

### 📌 std::

👉 `std::cout`

👉 இது:

* standard libraryல இருக்கும் cout

👉 `std::` = namespace

---

## 🔚 5. return 0;

```cpp
return 0;
```

---

### 🎯 Meaning:

👉 Program successஆ முடிந்தது

👉 Linuxல:

* 0 → success
* non-zero → error

---

## 🧩 Statement concept

👉 Statementன்னா:

👉 **ஒரு action**

---

### Example:

```cpp
std::cout << "Hello";
return 0;
```

👉 இரண்டும் statements
