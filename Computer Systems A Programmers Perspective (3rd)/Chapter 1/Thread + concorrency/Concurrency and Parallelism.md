# 🔁 3. Concurrency (exact definition)

Book:

> multiple, simultaneous activities

---

## 🧠 Deep meaning

👉 systemல:

```text
Task A
Task B
Task C
```

👉 எல்லாம் same timeல progress ஆகும்

---

## ⚠️ Important

```text
Simultaneous ≠ always truly same time
```

👉 single CPUல:

```text
A → B → A → B (fast switching)
```

👉 இது தான் concurrency

# 🚀 4. Parallelism (exact definition)

Book:

---

## 🧠 Meaning

```text

```

---

## 💥 Insight

```text
Concurrency = capability
```

# 🎯 5. Difference (deep clarity)

| Concept     | Meaning                          |
| ----------- | -------------------------------- |
| Concurrency | multiple activities exist        |
| Parallelism | those activities speed up system |

---

## 🧠 Example

👉 Concurrency only:

```text

```

👉 Parallelism:

```text
2 CPU cores → tasks truly run together
```
