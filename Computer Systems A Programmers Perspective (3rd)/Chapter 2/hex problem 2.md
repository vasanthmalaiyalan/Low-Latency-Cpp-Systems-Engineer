# 🧠 🔥 Practice Problem 2.3

👉 Table fill பண்ணணும்:
Decimal ↔ Binary ↔ Hex

---

## 🟢 Row 1 (given)

```text
0 → 00000000 → 0x00
```

---

## 🔵 Row 2

👉 Binary: `10101110`

### Step 1: Hex

```text
1010 1110
→ A    E
→ 0xAE
```

### Step 2: Decimal

```text
= 1×128 + 0×64 + 1×32 + 0×16 + 1×8 + 1×4 + 1×2 + 0×1
= 128 + 32 + 8 + 4 + 2
= 174
```

---

## 🟡 Row 3

👉 Binary: `00111100`

### Hex

```text
0011 1100
→ 3    C
→ 0x3C
```

### Decimal

```text
= 32 + 16 + 8 + 4
= 60
```

---

## 🔴 Row 4


```text
605C
+0005
-----
6061
```

✅ Answer: **0x6061**

---

## 🔵 B. 0x605c − 0x20

```text
605C
-0020
-----
603C
```

✅ Answer: **0x603C**

---

## 🟡 C. 0x605c + 32

👉 32 decimal = 0x20

```text
605C
+0020

607C
```

✅ Answer: **0x607C**

---

## 🔴 D. 0x60fa − 0x605c

