# 1️⃣ Simple Flashlight Circuit

முதலில் author ஒரு **very simple circuit** காட்டுகிறார்.

```
   +| |-
   Battery
      │
      │
     (Bulb)
      │
      │
     /   ← Switch
    /
   │
   └───────────────
```

### Explanation

Electricity flow path:

```
Battery → Wire → Bulb → Switch → Battery
```

Important rule:

```
Circuit complete → Bulb ON
Circuit break → Bulb OFF
```

Switch purpose:

```
open  → circuit break → light OFF
closed → circuit complete → light ON
```

---

# 2️⃣ Circuit is always a circle

Electric circuit must form a **closed loop**.

```
Battery
   │
   │
 Bulb
   │
 Switch
   │
   └──────── back to battery
```

If path breaks anywhere:

```
battery → wire → bulb → X
```

Light goes OFF.

# 3️⃣ What actually moves in circuit?

Author explains:

```
Electricity = movement of electrons
```

Electrons move through wire.

Like:

```
Atom → Atom → Atom → Atom
```

Electrons pass from one atom to another.

# 4️⃣ Atom structure

Author uses a simple model:

```
        electron
           ●
        /     \
       /       \
      ●  nucleus  ●
     /  (p,n)     \
    ●             ●
```

Inside atom:

| particle | charge |
| -------- | ------ |
| proton   | +      |
| electron | -      |
| neutron  | 0      |

Opposite charges attract:

```
+ attracts -
```

# 6️⃣ Battery inside flashlight

Battery converts:

```
chemical energy → electrical energy
```

# 7️⃣ Two batteries in series

Flashlights usually use **two batteries**.

Diagram:

```
  +| |-   +| |-
  Battery Battery
     │        │
     └────────┘
```

Voltage adds:

```
1.5V + 1.5V = 3V
```

So bulb gets stronger power.

---

# 8️⃣ Wrong battery direction

If battery reversed:

```
+| |- 
-| |+
```

Electrons cannot move properly.

Circuit fails.

Light OFF.

---

# 9️⃣ Parallel batteries

Another connection:

```
+| |-
+| |-
```

Parallel result:

```
Voltage = 1.5V
Battery life = longer
Brightness = lower
```

# 1️⃣1️⃣ Resistance concept

Resistance = difficulty for electrons to move.

Symbol:

```
/\/\/\/
```

Circuit diagram:

```
Battery ──/\/\/\/── Bulb
```

Unit:

```
Ohm (Ω)
```

Rules:

```
Long wire → more resistance
Thick wire → less resistance
```

---

# 1️⃣2️⃣ Voltage concept

Voltage = **potential energy of electricity**

Example author gives:

Brick example.

```
Brick on floor → low potential
Brick on building → high potential
```

Same:

```
Battery voltage = electrical potential
```

Unit:

```
Volt
```

Named after
Alessandro Volta

---

# 1️⃣3️⃣ Current

Current = number of electrons flowing.

Unit:

```
Ampere (Amp)
```

Named after
André-Marie Ampère

Example:

```
1 amp =
6,240,000,000,000,000,000 electrons per second
```

---

# 1️⃣4️⃣ Ohm's Law

Formula:

```
I = E / R
```

Where:

```
I = Current
E = Voltage
R = Resistance
```

Named after
Georg Simon Ohm

# 1️⃣7️⃣ Flashlight calculation

Flashlight example:

```
Voltage = 3V
Resistance = 4Ω
```

Using Ohm's law:

```
I = 3 / 4
```

Result:

```
Current = 0.75 Amp
```

Electrons per second:

```
4,680,000,000,000,000,000 electrons
```

---

# 1️⃣8️⃣ Power formula

Power:

```
P = E × I
```

Example:

```
3V × 0.75A = 2.25 watts
```

Unit named after
James Watt
